HDMI Cable Guide

Ultra High Speed HDMI

48Gbps - Required for:

  • 4K@120Hz with HDR
  • 8K@60Hz
  • VRR/ALLM gaming
  • eARC (full bandwidth)

Brands: Zeskit, Monoprice 8K, Cable Matters, Belkin Ultra

Premium High Speed HDMI

18Gbps - Sufficient for:

  • 4K@60Hz with HDR
  • 4K@120Hz SDR
  • Dolby Vision
  • ARC (not eARC)

Brands: Monoprice Certified, AmazonBasics Premium, BlueRigger

Fiber Optic HDMI

For long runs (10m+):

  • In-wall installations
  • Projector runs
  • No signal degradation
  • Directional (sourcedisplay)

Brands: RUIPRO, Monoprice SlimRun, Cable Matters Active

Port Connection Guide

Connection Port to Use Why
Source HDMI Processor RX0/RX1 (Input) Use RX0 for primary source (fastest switching)
HDMI Processor Display TX0 (Output) HDMI 1 TX0 for video, use display's primary HDMI port
HDMI Processor AVR TX1 (Output) HDMI IN eARC/ARC for audio return if supported
AVR Speakers Speaker terminals Use banana plugs for easy connection
